# Plum OPD Claim Adjudicator

An AI-powered tool that automates the adjudication (approve / reject / partial / manual-review) of Outpatient Department (OPD) insurance claims.

## Tech Stack
- Next.js 14 (App Router)
- React & Tailwind CSS
- Google Gemini API (Multimodal OCR & Reasoning)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000
